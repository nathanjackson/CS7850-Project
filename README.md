# CS7850 Project

Key-Value store based on Path ORAM.

## Quickstart

1. Create a virtual environment.

    ```
    virtualenv venv
    ```

2. Activate virtual environment

    ```
    . ./venv/bin/activate
    ```

3. Install dependencies.

    ```
    pip3 install -r requirements.txt
    ```

4. Run Tests

    ```
    ./tests/test_store.py
    ```
